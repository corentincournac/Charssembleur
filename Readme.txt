Pour créer un programme, écrivez simplement dans un fichier texte.
Pour exécuter un programme, lancer charssembleur avec le nom du fichier en paramètre (sous linux : ./charssembleur [fichier], sous windows, charsembleur.exe [fichier], ou déplacer le fichier sur l'exécutable dans l'interface graphique), ou lancez l'executable tout seul et entrer le nom du fichier ensuite. (sous linux : ./charssembleur. Sous windows : double clic sur charssembleur.exe)
N'oubliez pas de lire le manuel (manuel.pdf)

Vous trouverez la source de ce programme (c++) dans le dossier source, si vous utilisez un système d'exploitation exotique (par ex : mac OS X :P) et que vous avez besoin de recompiler.
